# Contributing to WebTestAgent

感谢你对 WebTestAgent 的关注！欢迎贡献代码。

## 关于 / About

WebTestAgent 是一个让测试变得像呼吸一样简单的 AI 驱动 Web 自动化测试工具。

WebTestAgent is an AI-powered web automation testing tool that makes testing as easy as breathing.

## 如何贡献 / How to Contribute

### 报告问题 / Report Issues

发现 bug 或有新功能建议？请提交 [Issue](https://github.com/your-repo/WebTestAgent/issues)。

Found a bug or have a feature request? Please open an [Issue](https://github.com/your-repo/WebTestAgent/issues).

### 提交代码 / Submit Code

1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送分支：`git push origin feature/your-feature`
5. 提交 Pull Request

1. Fork the repo
2. Create your branch: `git checkout -b feature/your-feature`
3. Commit your changes: `git commit -m 'Add some feature'`
4. Push to the branch: `git push origin feature/your-feature`
5. Submit a Pull Request

## 开发环境 / Development Setup

```bash
# 安装依赖 / Install dependencies
npm install && cd client && npm install

# 运行测试 / Run tests
npm test

# 启动开发服务器 / Start dev server
npm run dev
```

## 代码规范 / Code Style

- 使用 2 空格缩进
- Use 2 spaces for indentation
- 变量命名使用 camelCase
- Use camelCase for variable names
